## 설치
```bash

# 프로젝트 github에서 받음
git clone `` 
cd problembank-client 

# 필요하는 packge 설치
npm start

# .env 파일을 만들어서 환경변수를 추가함
vim .env 
REACT_APP_SERVER_API=http://localhost:3003
REACT_APP_SERVER_SCORE_API=http://localhost:5111
REACT_APP_CODE_SERVER=http://localhost

```
